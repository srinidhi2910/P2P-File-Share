A real-time peer-to-peer file sharing application built using WebRTC and Socket.io.  
This app enables direct browser-to-browser file transfer without storing files on the server.


Live Demo :-
https://p2p-file-share-8f2y.onrender.com/

---

Features

- 🔗 Create and join rooms
- ⚡ Real-time peer-to-peer connection (WebRTC)
- 📁 Direct file transfer between browsers
- 🧠 Signaling using Socket.io
- 🗂️ Transfer history stored in MongoDB
- 🔐 Secure room IDs using nanoid
- 👥 Room limit (max 2 users)


Tech Stack

- **Frontend:** HTML, CSS, JavaScript (EJS)
- **Backend:** Node.js, Express
- **Real-time:** Socket.io
- **P2P Communication:** WebRTC
- **Database:** MongoDB (Mongoose)
- **Deployment:** Render
